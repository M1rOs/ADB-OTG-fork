
# ADB>_OTG (ADB OTG with Termux Console)

This fork integrates a Termux-style terminal console UI for interactive ADB shell usage directly on Android.
(https://github.com/termux)

***>_ Run ADB commands without a computer [No Need ROOT] >_***



You can use ADB commands by connecting your Android smartphone to your smartphone.
You can use it only by installing the app without rooting or additional process.
